Calculadora Interativa

A calculadora interativa foi meu primeiro projeto utilizando JavaScript. Ela é capaz de realizar as quatro operações matemáticas básicas: adição, subtração, multiplicação e divisão.

O desenvolvimento deste projeto me ensinou conceitos importantes como:

  • Manipulação do DOM para capturar eventos de clique
  • Uso de funções para organizar o código
  • Lógica de programação para realizar os cálculos
  • Validação de entrada para evitar erros

A interface foi construída com HTML e CSS, enquanto toda a funcionalidade foi inovadora com JavaScript puro, sem o uso de bibliotecas externas.

Blog de Tecnologia

Este projeto de blog foi desenvolvido para praticar o uso do framework Bootstrap e conceitos de design responsivo. O blog contém artigos sobre tecnologia e programação, organizados em um layout limpo e fácil de navegar.

Principais características inovadoras:

  • Layout responsivo que se adapta a diferentes tamanhos de tela
  • Barra de navegação com menu de navegação
  • Sistema de grade para organizar os artigos
  • Cartões para exibir resumos dos posts
  • Formulário de contato funcional

Através deste projeto, pude aprender sobre os componentes do Bootstrap e como utilizá-los para acelerar o desenvolvimento do front-end sem perder a qualidade do design.
